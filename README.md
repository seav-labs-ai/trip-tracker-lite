# Trip Tracker - Family Travel Guide

A curated, mobile-first web application built to serve as a digital hub for family travel. Originally designed for a real two-week family trip, this repository contains a genericized, portfolio-ready version of the application.

## 🌟 Overview

When traveling internationally with family, keeping track of itineraries, safety information, restaurant recommendations, and maps in one central location is critical. This application solves that problem by providing a fast, beautifully designed, password-protected portal.

### Key Features
*   **Curated Guides:** Structured sections for Food, Activities (Info), and Safety guidelines.
*   **Interactive Mapping:** Embedded maps with important locations pre-pinned.
*   **Mobile-First Design:** Optimized for on-the-go access via mobile browsers.
*   **Authentication Flow:** Simple password-protected access using Next.js middleware and cookies.
*   **Modern UI:** Sleek, responsive interface with gradient text and modern styling.

## 🛠️ Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Styling:** CSS Modules / Vanilla CSS
*   **Language:** TypeScript
*   **Deployment:** Vercel

## 🚀 Getting Started

### Local Development

1.  Clone the repository:
    ```bash
    git clone https://github.com/seav-labs-ai/trip-tracker-lite.git
    cd trip-tracker-lite
    ```
2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
3.  Set up your environment variables based on the authentication requirements (if any local overrides are needed).
4.  Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
